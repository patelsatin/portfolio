# Portfolio Website

This is my personal portfolio website, built with **React, SCSS, and Firebase**.
It showcases my work, projects, and experience as a software engineer.

## 🚀 Live Demo

[Visit Website](https://portfolio-2b6ce.web.app/) <!-- Replace with actual hosted link -->

## 🛠️ Tech Stack

* React (Frontend framework)
* SCSS (Styling)
* Firebase (Hosting & Storage)

## 📂 Project Structure

* `/src` → React components, styles, and logic
* `/public` → Static assets (HTML, images, icons)
* `firebase.json` → Firebase hosting configuration
* `.github/workflows` → GitHub Actions CI/CD setup

## ⚙️ Getting Started

### Prerequisites

* Node.js (>= 16)
* Firebase CLI (`npm install -g firebase-tools`)

### Setup

```bash
# Clone repo
git clone https://github.com/patelsatin/portfolio.git
cd portfolio

# Install dependencies
npm install

# Run locally
npm start

# Build for production
npm run build

# Deploy to Firebase
firebase deploy
```
